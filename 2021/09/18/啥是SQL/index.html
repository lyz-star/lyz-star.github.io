啥是SQL

Created2021-09-18|Updated2021-09-22|MySQL
|Post View:

1.1 数据库

​ 数据库就是一个文件柜，用于存放数据。

数据库（database）
保存有组织的数据的容器（通常是一个文件或一组文件）。

1.2 表

​ 当我们往文件柜中存放资料时，并非随手乱扔。而是有特定的结构。我们会创建一个表。

表（table）
某种特定类型数据的结构化清单。

​ 通常我们在同一张表中存放的是同一种类型的数据。比方说我们不应该将用户数据和购买记录放在同一张表中。并且表的名字在同一个数据库中是唯一的。

​ 表具有一定的结构。表的结构定义的表中的数据会如何存储，存储什么样的数据，数据如何被分解。用来描述表的信息就是模式。

模式（schema）
关于数据库和表的布局结构的信息。

1.3 列

​ 表是由不同的列组成的。列存储表中的某一部分的信息（字段）。

列（column）

表中的字段。表一般是由多个列组成的。每一列存储字段定义的信息。等同于表格中的列。

​ 正确的将数据分解十分必要。例如，姓名，年龄，地址等都应该是独立的列。通过他们独立的列，在进行数据分类过滤将非常有帮助。在实际应用中要根据情况合理的分割数据。

列中的数据要注意他的数据类型兼容。每个列都有他自己的数据类型。

1.4 行

​ 表格中的数据是按照行存储的。一行数据为相关联的一组数据。

1.5 主键

​ 表中的一行数据用于唯一标识的字段，即可作为主键存在。例如身份证号码，订单id号码。

主键（primary key）
能够唯一标识表中每一行。

虽然并不总是需要主键，但多数数据库设计者都会保证他们创建的每表具有一个主键，以便于以后的数据操作和管理。

1.6 什么是SQL

SQL是 Structured Query Language（结构化查询语言）的缩写。SQL是一种专门用来与数据库沟通的语言。

SQL有如下的优点：

  • SQL不是某个特定数据库供应商专有的语言。几乎所有重要的 DBMS都支持 SQL，所以学习此语言使你几乎能与所有数据库打交道。
  • SQL简单易学。它的语句全都是由有很强描述性的英语单词组成，而且这些单词的数目不多。
  • SQL虽然看上去很简单，但实际上是一种强有力的语言，灵活使用其语言元素，可以进行非常复杂和高级的数据库操作。
